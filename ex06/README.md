# 第6回
## ゲーム開発演習
### 内容
- ホッケーゲーム
- 味方の台と敵の台がステージ上にある一つのボールを打ち返し合う
- 味方の台は上下方向のみ矢印キー上下で動かすことが出来る
- 敵の台は上下方向のみ動き、ボールを追いかけるように動く。ボールと跳ね返る時に少しづつ速度を上げるようにする
- ボールは台と跳ね返る時に少しづつ速度を上げるようにする。スコアに応じて、初期の位置をランダムに設定する
- 経過時間や味方と敵のスコアを計算する
